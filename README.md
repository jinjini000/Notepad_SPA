# 싱글 페이지 어플리케이션 연습

React를 활용하여 만든 메모장SPA 입니다.
로케이션스토리지를 활용하여 메모와 관련된 내용을 json형태로 저장하고 활용하는 코드입니다.
검색이 가능하며, 제목과 내용을 동시에 검색합니다.
메모리스트를 클릭하면 세부사항을 볼 수 있습니다.

## 설치 & 사용법 (Getting Started)  

### 필요 환경  
- Node.js (권장)  
- npm 또는 yarn  

### 설치  
```bash
git clone https://github.com/jinjini000/Notepad_SPA.git
```
```bash
cd Notepad_SPA
```
```bash
npm install
```

### 실행
```bash
npm run dev
```
http://localhost:5173/ 접속
